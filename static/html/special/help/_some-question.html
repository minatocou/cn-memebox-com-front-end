常见问题

一、购物流程

  • 登录/注册 → 挑选商品 → 加入购物车 → 确认订单 → 支付订单 → 收货与评价。

二、新用户注册

  1. 首先点击右下角“我哒”，进入登录页面后点击右上角“注册”。
  2. 请根据提示在信息栏内正确填入注册信息，填写完成后点击下方的“注册”按钮。

三、登录

  1. 点击右下角“我哒”，进入登录页面。
  2. 在登录页面的信息栏内填入对应信息（注册手机号及密码），点击“登录”按钮进行登录，登录成功后，系统将自动跳转至美美箱首页。

四、忘记密码

  • 请进入登录页面→点击“忘记密码”跳转到重置密码页面→输入用于登录的手机号，点击“提交”→我们将会发给您重置密码的链接网址。或使用手机号快捷登录。

五、购买须知

  1. 官网、商品咨询
    1. 请点击APP右下角『我哒』，您可以根据目录查看需要知道的账户信息。例如：查看订单、查看蜜豆、蜜米等级、优惠券使用情况、修改/添加收货地址等等。
    2. 服务时间为周一至周日：9:00-21:00（遇到法定节假日，服务时间另行安排通知）。
    3. 请您进入我的账户，点击“订阅邮件”，勾选“订阅申请”，这样就成功订阅了美美箱的活动及优惠信息；反之则取消订阅。
  2. 订单&支付&更改&取消&缺货
    1. 订单确认生成后，请在20分钟内支付，过时系统将自动取消订单；
      系统自动取消的订单虽支付宝“账单”中显示“等待付款”，若您重新在支付宝中支付钱款，订单无法恢复正常，此时只能联系客服为您办理退款。
    2. 购买前请注意，一旦结算完成后商品将不能更换；
      直邮订单确认生成后，无法对产品、收货地址、姓名、收货电话做任何修改；请仔细填写收货信息，一旦发生错误，往返运费及相关后果需由您自行承担。
    3. 当天订购可立即取消订单，需要您在app中点击取消订单办理退款，取消后优惠券不返还、现金券返还；
      订单发货后将不能取消；优惠券、现金券均不返还。
    4. 如果您的订单中有商品缺货，客服会通过电话、短信通知您，为您处理；
      若您的电话始终无人接听；或者收到短信未及时与我们联系处理；订单将为您保留3个工作日；
      超过3个工作日还未处理的订单，系统将自行取消订单，退款将在1-3个工作日内由原路退回。
    5. 本网站产品订购是针对个人顾客进行购买，故若经过核实发现相同ID注册多个帐号或相同ID、收货人姓名、电话、地址相同订购了多张不符合优惠条件的订单，系统将自动取消订单不予以发货；请广大会员理解！
      若本网站系统发生异常问题，如：造成您反复不断购买，生成多张异常订单，系统将自动取消订单不做另行通知，并在3个工作日将款项由原路退回；若您使用了相关优惠我们也会返还给您。
